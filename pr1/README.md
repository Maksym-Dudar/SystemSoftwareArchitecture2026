## Опис

Ця програма реалізована на мові C. Першим зчитує число для системи численнята. Другим зчитує саме число і в кінці переводить це число в десяткову систему численн.
Підтримується дробова частина, яка вводиться через кому `,`.

---

## Приклад роботи

```
Введіть систему числення: 2
Введіть число: 101,01
5.250000
```

---

## Обмеження

* Максимальна довжина числа — **99 символів**
* Підтримуються лише **цифри 0–9**
* Літери **не підтримуються**

---

## Збірка та запуск

```bash
gcc main.c -o main 
./main
```
